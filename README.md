# HR Employee Attrition Dashboard

## Overview
This project was developed using Tableau as part of the Data Analyst Internship at SkillCraft Technology.

The dashboard analyzes employee attrition patterns across departments, job roles, and age groups.

## Dashboard Metrics
- Total Employees: 1,470
- Attrition Count: 237
- Average Age: 36.92

## Key Insights
- Research & Development recorded the highest attrition (133 employees).
- Sales recorded the second-highest attrition (92 employees).
- Human Resources had the lowest attrition (12 employees).
- Employees aged 25–35 showed the highest attrition.
- Laboratory Technicians experienced the highest attrition among job roles.

## Tools Used
- Tableau Public
- HR Employee Attrition Dataset
